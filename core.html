const AvmType *AvmObjectGetType(object self)¶

Gets information about the type of an object.

You should use this function to get the type of an object. This assumes that the ._type field is correctly initialized by the object constructor.

This function returns the actual runtime type of an object bypassing any base types. That is, for an object of type T where T inherits from another type B, this function will always return typeid(T) regardless if the object is passed as a T* or a B*.

To check if an object is an instance of a type (possibly derived) use the instanceof macro.

To handle the returning AvmType instance, you may need to include typeinfo.h

This function is not overridable.

Parameters

self – The object instance.

Pre

Parameter self must be not null.

Returns

The type information of the object.

bool AvmObjectEquals(object self, object other)¶

Compares two objects for equality.

This function compares 2 objects for value equality, that is both objects in-memory representation must be exactly the same for this function to return true.

Overriding this function may be especially desirable for non-trivial types.

This function is overridable by the FnEntryEquals entry in the VFT.

Parameters

  • self – The first object.

  • other – The second object.

Pre

Parameter self must be not null.

Pre

Parameter other must be not null.

Returns

true if the two objects are equal, otherwise false.

void AvmObjectFinalize(object self)¶

Finalizes an object.

This function calls AvmObjectSurpressFinalizer to make sure that the finalizer is not called more than once.

This function is overridable by the FnEntryFinalize entry in the VFT.

Overriding should be done when an object needs to release non-memory resources.

Parameters

self – The object instance.

Pre

Parameter self must be not null.

void AvmObjectSurpressFinalizer(object self)¶

Disables the finalizer of an object.

When called makes the runtime ignore finalization of the object. This may be useful if writing functions that deterministically finalizes an object.

This function is not overridable.

Parameters

self – The object instance.

Pre

Parameter self must be not null.

int AvmRuntimeInit(int argc, str argv[], AvmEntryPoint entry)¶

Initializes the Avium runtime and calls into the entry point.

Many Avium functions cannot be called without initializing the runtime first.

When using the whole framework, this function is called automatically.

Parameters

  • argc – The argc parameter from main.

  • argv – The argv parameter from main.

  • entry – The entry point.

Returns

A status code such as EXIT_SUCCESS or EXIT_FAILURE.
