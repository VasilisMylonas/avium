enum AvmSeekOrigin¶

Defines the origin point for a seek operation.

Values:

enumerator SeekOriginBegin¶

Seek from the beginning of the stream.

enumerator SeekOriginCurrent¶

Seek from the current stream position.

enumerator SeekOriginEnd¶

Seek backwards from the end of the stream.

AvmError *AvmStreamSeek(AvmStream *self, _long offset, AvmSeekOrigin origin)¶

Seeks to a position in a stream.

If origin is SeekOriginBegin, then the position is set to offset. If origin is SeekOriginCurrent, then the position is incremented, or decremented according to offset. If origin is SeekOriginEnd, then the position is incremented, or decremented according to offset.

Parameters

  • self – The AvmStream instance.

  • offset – The offset.

  • origin – The origin point for seek operation.

Pre

Parameter self must be not null.

Returns

The result of the IO operation.
