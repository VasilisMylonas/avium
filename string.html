AVM_CLASS (AvmString, object, { uint _capacity;uint _length;char *_buffer;})

A dynamic heap-allocated string.

AvmString AvmStringNew(uint capacity)¶

Creates an AvmString with a specified capacity.

The capacity is the number of characters that the string will be able to hold until it needs to allocate more memory. If 0 is used as the capacity, then no memory is allocated until a character is pushed.

Parameters

capacity – The capacity of the AvmString.

Returns

The created instance.

AvmString AvmStringFrom(str contents)¶

Creates an AvmString from a raw string.

The raw string is copied into a heap buffer. The capacity of the resulting string may be larger, in order to reduce total reallocations.

Parameters

contents – The contents of the AvmString.

Pre

Parameter contents must be a ‘\0’ terminated string.

Pre

Parameter contents must be not NULL.

Returns

The created instance.

void AvmStringPushChar(AvmString *self, char character)¶

Pushes a character to the end of an AvmString.

If the AvmString’s length is less than its capacity, then the character is simply appended to the end, otherwise the AvmString allocates new memory. The allocated memory may be larger than one character, in order to reduce total reallocations.

Parameters

  • self – The AvmString instance.

  • character – The character to push.

Pre

Parameter self must be not NULL.

void AvmStringPushStr(AvmString *self, str contents)¶

Pushes a str to the end an AvmString.

If the AvmString’s length is not sufficient to hold the str then the AvmString allocates new memory. The allocated memory may be larger than the length of the str, in order to reduce total reallocations.

Parameters

  • self – The AvmString instance.

  • contents – The str to push.
